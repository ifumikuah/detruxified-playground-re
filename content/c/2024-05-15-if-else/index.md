If else di C.

```c
if (bool_eval) {
  // Execute this block
}
```

```c
if (bool_eval) {
  // Execute this block
} else {
  // Execute this block
}
```

```c
if (bool_eval) {
  // Execute this block
} else if (bool_eval_2) {
  // Execute this block
} else {
  // Execute this block
}
```

Operator seperti biasa:

- `>` atau `>=`
- `<` atau `<=`
- `==` dan `!=`
