### Sinopsis

```js
const calculateArea = (width, height) => {
    const area = width * height;
    return area;
}
```

### Details

Tidak perlu lagi membuat `function` sesudah variable, cukup tambahkan arrow `=>`.

```js
const triangle = (base, height) => {
  area = 0.5 * (base * height);
  return area;
}
```
