### Synopsis

```js
if (morbius == true) {
  console.log("It's Morbin Time!!");
}
```
### Details

logika *if*, jika kondisi memenuhi kriteria if (`criteria`), maka blok tersebut akan di eksekusi

```js
let drMorbiusMorbin = true;

if (drMorbiusMorbin) {
    console.log("It's Morbin Time!!");
}
```
Output
```plain
It's Morbin Time!!
```
