## Sinopsis

```js
let w = 4;
w = w + 1;
```

### Detail

Menggabungkan variabel dengan operator aritmatika

```js
let x = 4;
x = x + 1;

console.log(x);
```
Output
```cmd
5
```

Cara singkat jika ingin tetap menggunakan variabel yang sama


```js
let angka = 10;
console.log(angka);

angka += 5;
console.log(angka);
```
Output
```cmd
10
15
```

Operator aritmatika yang bisa digunakan adalah : `+`,`-`,`*`,`/`
