### Sinopsis

```js
console.log("Roti" + "Tawar");
```

### Details

Menggabungkan antar string dengan operator `+`.

```js
console.log("Ikan" + "Lele")
```
Output
```cmd
IkanLele
```

Spasi juga dihitung sebagai karakter

```js
console.log("Ikan " + "Lele")
//atau
console.log("Ikan" + " " + "Lele")
```
Output
```cmd
Ikan Lele
Ikan Lele
```
