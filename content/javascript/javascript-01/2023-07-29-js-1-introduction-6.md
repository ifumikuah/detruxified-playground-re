### Synopsis

```js
console.log('Hello'.length);
```

### Details

Print jumlah karakter dalam satu string

```js
console.log('Bye'.length);
```
Output
```cmd
3
```

Ingat bahwa spasi juga dihitung sebagai karakter
```js
console.log('Kura Kura'.length);
```
Output
```cmd
9
```
