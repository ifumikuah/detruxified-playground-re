Fungsi matematika menghandle angka dengan beberapa fungsi matematika di pemrograman pada umumnya.

```php
  <?php
    $rounded = round(5.78);
    echo "Rounded 5.78 is {$rounded} <br>";

    echo sqrt(144);
  ?>
```
