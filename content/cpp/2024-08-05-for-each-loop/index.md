Perulangan khusus objek iterable seperti array.

```cpp
  int nums[] = {1, 2, 3, 4, 5, 8};

  for (int num : nums)
    std::cout << num;
```

Disini `num` adalah variable yang dideklarasi untuk element individual yang ada di array `nums`.
